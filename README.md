# 📖 My English Study — PWA

## 파일 구성
| 파일 | 설명 |
|------|------|
| `index.html` | 메인 앱 (전체 UI) |
| `manifest.json` | PWA 설정 (앱 이름, 아이콘, 색상) |
| `sw.js` | Service Worker (오프라인 캐싱) |
| `icon-192.png` | 홈 화면 아이콘 (소) |
| `icon-512.png` | 홈 화면 아이콘 (대) |

## 배포 방법 (GitHub Pages 추천)

1. GitHub에서 새 저장소 생성
2. 이 폴더의 **모든 파일 5개**를 업로드
3. Settings → Pages → Branch: main → Save
4. `https://사용자명.github.io/저장소명` 접속
5. 크롬: "홈 화면에 추가" 배너 탭
6. Safari: 공유(□↑) → "홈 화면에 추가"
